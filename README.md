# How to use
  - Grab the empty ESL pdf from Hanshow
  - Upload this file in the left upload-box
  - Manually enter all Aisle names and corresponding groups in the right section, or scrape these from Momo.
    - To scrape from Momo:
    - Install the extention listed in the sourcecode
    - Go to 'Vulpad' in Momo
    - Open the extention and press 'Get'
    - Click back in Momo, to ensure it has copy-permissions for that tab
    - Wait untill it says "Everything copied"
    - Paste output in the big-textbox
    -  Click Parse
  -  Then select all ailes you want to get
  -  If you want unlisted groups to still show up, check the box at the bottom. These will then appear at the start of the generated document
  -  Click Process
  -  A new tab should open after the input has been processed
  -  Press ctrl+p and print
